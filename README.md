# dotfiles

These are my .dotfiles, there are many like it, but these are mine.

## Install

``` sh
git clone https://github.com/mike-prince/dotfiles.git $HOME/.dotfiles
zsh $HOME/.dotfiles/install.sh $HOME/.dotfiles
exec zsh
```

If you don't supply a path to *dotfiles* as an argument to the install script it will default to `$HOME/.dotfiles` anyway.
